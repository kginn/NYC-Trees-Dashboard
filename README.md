# NYC Tree Census Visualization

[![Live Dashboard](https://img.shields.io/badge/View_Live_Dashboard-Click_Here-27AE60?style=for-the-badge)](https://nyc-trees-dashboard.onrender.com)

## Overview
This interactive dashboard provides a comprehensive analysis of New York City's urban forest. It allows users to explore how community stewardship, geographic location, and species type impact the overall health and distribution of trees across the boroughs. 

## Tools Used
* **Language:** Python
* **Framework:** Dash (Plotly)
* **Data Manipulation:** Pandas
* **Deployment:** Render (Gunicorn/Flask)

## Key Features & Analysis
* **Dynamic KPIs:** Instantly calculates total trees, percentage in good health, average diameter, and dominant species based on active filters.
* **Geospatial Mapping:** Utilizes interactive Mapbox scatter plots to visualize tree health at precise coordinates across different boroughs.
* **Stewardship Impact:** A stacked bar chart analyzing whether community care (stewards) correlates with higher percentages of healthy trees.
* **Species vs. Size Correlation:** A scatter plot evaluating the relationship between a tree's average trunk diameter and its overall health.
* **Interactive Filtering:** Users can drill down by borough, specific health statuses, and trunk diameter ranges to isolate specific environmental trends.
